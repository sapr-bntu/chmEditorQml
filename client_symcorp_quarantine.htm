Карантин: Просмотреть карантин

Иногда клиент обнаруживает неизвестный вирус, который не удается исправить с помощью текущего набора описаний вирусов. В других ситуациях пользователь может подозревать, что файл заражен, хотя клиент не находит вирусов. Карантин надежно помещает в карантин потенциально зараженные файлы от остальной системы. Вирус не может распространяться из карантина.

В результате перемещения файла в карантин вирус не удаляется. Он остается на компьютере до тех пор, пока пользователь не удалит или не исправит зараженный файл.

Примечание:

Карантин не применяется для загрузочных вирусов. Обычно загрузочные вирусы заражают загрузочный сектор или таблицу разделов, а эти элементы переместить в карантин нельзя.

Табл.: Параметры карантина

Элемент

Описание

Угроза

Тип угрозы, которую несет в себе вирус.

Имя файла

Имя зараженного файла.

Примечание:

Операционные системы на некоторых языках могут неправильно интерпретировать некоторые символы в названиях вирусов. Если символы не удается интерпретировать, они отображаются в уведомлениях как знаки вопроса. Например, в именах некоторых вирусов, выраженных в юникоде, есть двухбайтовые символы. Если клиент работает на компьютере с английской операционной системой, такие символы будут показаны в виде знаков вопроса.

Тип

Тип зараженного файла.

Исходный путь

Исходный каталог, из которого зараженный файл был перемещен в карантин.

Состояние

Состояние файла, помещенного в карантин. Например, одно из возможных состояний - "Заражен".

Дата

Дата перемещения файла в карантин.

Восстановить

Восстановить выбранный файл в исходном каталоге.

Удалить

Полностью удалить выбранный файл.

Заново просканировать все

Пытается исправить выбранные файлы после обновления описаний вирусов.

Экспортировать

Позволяет экспортировать список объектов в карантине в формате с разделителями-запятыми (.csv) или в формате базы данных Access (.mdb).

Добавить

Вручную добавить файл в карантин. Нажмите "Добавить", чтобы выбрать каталог и указать файл, который необходимо переместить в карантин.

Отправить

Для отправки выбранного файла на анализ в Symantec Security Response следуйте показанным на экране инструкциям.

Параметры очистки

Эти параметры управляют расписанием очистки файлов.
