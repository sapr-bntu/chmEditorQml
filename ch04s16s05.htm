Удаление файлов из карантина

Из карантина можно вручную удалить файлы, которые больше не нужны. Можно также задать период времени, через который файлы будут удаляться автоматически.

Примечание:

Администратор может задать количество дней, в течение которых файлы могут храниться в карантине. По истечении этого времени файлы автоматически удаляются из карантина.

См. Автоматическое удаление файлов из карантина.

Как вручную удалить файлы из карантина

  1. На боковой панели клиента выберите Просмотреть карантин.

  2. Выберите один или несколько файлов.

  3. Нажмите кнопку Удалить.
