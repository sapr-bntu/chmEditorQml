Отправка потенциально зараженного файла в группу Symantec Security Response для анализа

В некоторых случаях клиенту не удается удалить вирус из файла. У вас также может возникнуть подозрение, что некоторый файл заражен, хотя клиент не обнаруживает заражения. Такой файл можно передать в группу Symantec Security Response для проверки на наличие заражения. Для передачи образца должно быть настроено подключение к Интернету.

Примечание:

Параметр Отправить в Symantec Security Response не будет доступен, если администратор запретил эту операцию.

Как отправить файл из карантина в группу Symantec Security Response

  1. На боковой панели клиента выберите Просмотреть карантин.

  2. Выберите файл в списке.

  3. Нажмите кнопку Отправить.

  4. Следуйте инструкциям мастера по сбору нужной информации и передаче файла для анализа.
