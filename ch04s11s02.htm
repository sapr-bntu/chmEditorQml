Советы по выбору вторых действий для угроз безопасности

При выборе второго действия для угроз безопасности следует учесть требуемый уровень контроля над файлами. Если на компьютере не создается резервная копия важных данных, не следует использовать действие «Удалить угрозу». Хотя это действие позволяет устранить угрозу безопасности, потенциально оно может привести к сбою какого-либо другого приложения. Вместо этого действия рекомендуется выбрать действие «Поместить в карантин угрозу», для того чтобы при необходимости можно было отменить изменения, сделанные клиентом.
