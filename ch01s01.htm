Сведения о клиенте

Компания Symantec предоставляет два описанных ниже продукта для защиты конечных точек, которые могут использоваться совместно или по отдельности.

Табл.: Типы клиентов для защиты конечных точек

Тип клиента

Описание

Symantec Endpoint Protection

Программа Symantec Endpoint Protection защищает компьютер от угроз из Интернета и угроз безопасности.

См. Сведения о клиенте Symantec Endpoint Protection.

Symantec Network Access Control

Программа Symantec Network Access Control гарантирует, что параметры безопасности компьютера соответствуют политикам, принятым в сети. В число параметров безопасности могут входить названия установленных программ, параметры этих программ, файлы сигнатур, исправления и другие элементы.

См. Сведения о клиенте Symantec Network Access Control.

Администратор установил один из этих продуктов Symantec или оба продукта на данный компьютер. Если клиент был установлен администратором, то администратор указал, какие продукты должны быть включены. Администратор должен сообщить вам, какие задачи потребуется выполнять с помощью клиента.

Если клиент установлен на отдельном компьютере, такая установка называется автономной. Работой автономного программного обеспечения клиента не управляет администратор.

См. О клиентах, управляемых централизованно, и клиентах под автономным управлением.

Примечание:

Если администратор установил на компьютер только один из этих продуктов, то его название будет показано с строке заголовка. Если установлены оба продукта, в строке заголовка будет показано "Symantec Endpoint Protection".
