Добавить исключения угроз безопасности по расширению

Это окно позволяет указать типы файлов, которые следует исключить из сканирования. Расширения необходимо удалять по одному. Расширения могут содержать пробелы.

Если необходимо добавить несколько расширений, введите одно расширение и добавьте его. Затем повторите операцию для каждого расширения, которое необходимо добавить. Не указывайте в поле Добавить несколько расширений. Если указано несколько расширений, клиент будет рассматривать их как одно длинное расширение.

Табл.: Исключенные расширения

Параметр

Описание

Добавить

Позволяет добавить расширение в список расширений, которые должны быть исключены из сканирования.

Примечание:

Не указывайте перед расширением точку (.). Например, вместо ".exe" следует указать просто "exe".

Удалить

Позволяет удалить расширение из списка.
