Настройка размера журналов защиты от угроз из сети и управления клиентами

Для каждого журнала защиты от угроз из сети и управления клиентами можно указать размер журнала.

Как изменить размер журналов

  1. На странице "Состояние" клиента нажмите Параметры справа от пункта "Защита от угроз из сети", а затем выберите Изменить параметры.

  2. В окне "Параметры защиты от угроз из сети" откройте вкладку Журналы и укажите максимальный размер файла журнала в килобайтах в текстовом поле "Максимальный размер файла журнала".

    Для экономии памяти компьютера размер журнала должен оставаться небольшим. Значение по умолчанию для всех журналов, за исключением журнала управления и журнала пакетов, составляет 512 КБ. Для журнала пакетов и журнала управления значение по умолчанию составляет 1024 КБ.

  3. Нажмите кнопку OK.
