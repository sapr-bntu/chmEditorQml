Экспорт и импорт правил

Правила можно передать на другой клиент, чтобы их не пришлось создавать заново. Для этого правила необходимо экспортировать с одного компьютера и импортировать на другой. При импорте правила добавляются в конец списка правил. Импортированные правила не заменяют существующие, даже если импортированное правило совпадает с существующим.

Экспортированные и импортируемые правила хранятся в файлах .sar.

См. Добавление правила брандмауэра.

Как экспортировать правила

  1. На боковой панели клиента выберите Состояние.

  2. В разделе "Защита от угроз из сети" выберите Параметры > Настроить правила брандмауэра.

  3. В окне "Настроить правила брандмауэра" выберите правила, которые необходимо экспортировать.

  4. Щелкните на правиле правой кнопкой мыши и выберите команду Экспортировать выбранные правила.

  5. В окне "Экспорт" укажите имя файла, затем нажмите Сохранить.

  6. Нажмите кнопку ОК.

Как импортировать правила

  1. На боковой панели клиента выберите Состояние.

  2. В разделе "Защита от угроз из сети" выберите Параметры > Настроить правила брандмауэра.

  3. В окне "Настроить правила брандмауэра" щелкните правой кнопкой на списке правил и выберите команду Импортировать правило.

  4. В окне "Импорт" найдите файл .sar, содержащий импортируемые правила.

  5. Нажмите кнопку Открыть.

  6. Нажмите кнопку ОК.
