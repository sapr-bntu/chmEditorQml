Сведения о работе с зараженными файлами в карантине

После того как файл перемещен в карантин, можно выполнить любое из следующих действий:

  • Восстановить выбранный файл в исходном каталоге.

  • Полностью удалить выбранный файл.

  • Повторно просканировать файлы после получения обновленных описаний вирусов.

  • Экспортировать содержимое карантина в формате с разделителями-запятыми (*.csv) или в формате базы данных Access (*.mdb).

  • Вручную добавить файл в карантин. Можно выбрать каталог и указать файл, который необходимо переместить в карантин.

  • Отправить файл в Symantec Security Response. Для отправки выбранного файла на анализ следуйте показанным на экране инструкциям.

См. Управление карантином.
