Включение интеллектуальной фильтрации трафика

Интеллектуальная фильтрация трафика позволяет производить обычный обмен конкретных сетевых служб без явного разрешения этих служб с помощью правил. В ходе обработки эти правила срабатывают до правил брандмауэра, поэтому любой пакет, прошедший обработку встроенным правилом, будет разрешен. Встроенные правила можно настроить для служб DHCP, DNS и WINS. Запрос этих служб должен быть отправлен компьютером клиента, и ответ сервера на него должен быть получен в течение предустановленного пятисекундного интервала. Этот ответ проверяется на допустимость для исходного клиентского запроса.

Интеллектуальные фильтры пропускают пакет в случае, если был сделан запрос. Они не блокируют пакеты. Пакеты разрешаются и блокируются с помощью правил брандмауэра.

См. Включение параметров трафика и скрытого просмотра веб-страниц.

Как включить интеллектуальную фильтрацию трафика

  1. На боковой панели клиента выберите Изменить параметры.

  2. В разделе Защита от угроз из сети нажмите кнопку Изменить параметры.

  3. В окне Параметры защиты от угроз из сети выберите Брандмауэр.

  4. Отметьте любые из следующих переключателей:

    • Включить Smart DHCP

    • Включить Smart DNS

    • Включить Smart WINS

  5. Нажмите кнопку OK.
