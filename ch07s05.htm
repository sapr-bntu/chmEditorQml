Просмотр журналов Network Access Control

Клиент Symantec Network Access Control использует в работе следующие журналы для мониторинга различных аспектов работы и для проверки целостности хоста:

Журнал безопасности

Содержит сведения о результатах и состоянии проверки целостности хоста.

Системный журнал

Содержит сведения о всех операционных изменениях клиента, таких как подключение к серверу управления и обновление политики безопасности клиента.

На управляемых клиентах оба упомянутых выше журнала могут периодически отправляться на сервер. Содержимое журналов помогает администратору проанализировать общее состояние безопасности сети.

Записи журналов можно экспортировать.

Как просмотреть журналы Symantec Network Access Control

  1. На боковой панели клиента выберите Состояние.

  2. Для того чтобы просмотреть системный журнал, в разделе Network Access Control выберите Параметры > Показать журналы.

  3. Для того чтобы просмотреть журнал безопасности, в окне "Журналы управления клиентом - Системный журнал" выберите Показать > Журнал безопасности.

  4. Выберите Файл > Выход.
