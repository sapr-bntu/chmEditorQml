Устранение неполадок: Учетная запись Windows

На этой панели показан профиль текущего пользователя.

Примечание:

Используйте ее только по указанию специалистов службы поддержки Symantec.

Табл.: Учетная запись Windows

Параметр

Описание

Пользователь

Имя, домен и SID текущего пользователя системы.

Группы

Список имен, доменов и атрибутов групп Active Directory, в которые входит пользователь.

Привилегии

Список имен, отображаемых имен и атрибутов привилегий Active Directory либо действий, которые пользователю разрешено выполнять.
