Параметры защиты от угроз из сети: Сеть Microsoft Windows

На этой вкладке можно указать сетевой адаптер и права на просмотр сетевых ресурсов.

Табл.: Параметры сети Microsoft Windows

Параметр

Описание

Выберите настраиваемый адаптер

Задает сетевую карту, через которую передается сетевой трафик.

Показать папки и принтеры в сети

Позволяет просматривать другие компьютеры, устройства и принтеры, подключенные к сети.

Примечание:

Если этот параметр не будет выбран, то файлы нельзя будет копировать с сетевых дисков.

Разрешить общий доступ к моим папкам и принтерам в сети

Разрешает другим пользователям сети просматривать папки и файлы, расположенные на вашем компьютере.

Блокировать трафик сети Microsoft Windows во время работы экранной заставки

Автоматически блокирует весь трафик сетевого окружения, когда на компьютере работает экранная заставка. Когда работа с компьютером возобновляется, восстанавливается прежний уровень защиты.
