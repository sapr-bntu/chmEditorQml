Параметры для отдельного приложения

Администратор может настроить параметры приложений, которые были запущены с момента запуска службы клиента или запросили разрешение подключиться к сети.

Для приложения можно настроить ограничения, например, указав разрешенные IP-адреса и порты приложения. Можно просмотреть и изменить действие, выполняемое клиентом для каждого приложения, которое пытается получить доступ через сетевое соединение. Таким образом можно создать правило брандмауэра для отдельного приложения.

Примечание:

Если параметры приложения противоречат правилу брандмауэра, то правило брандмауэра считается более приоритетным. Например, правило брандмауэра, блокирующее весь трафик между 1:00 и 8:00 утра, переопределяет расписание определенного приложения видеосвязи.

См. Добавление правила брандмауэра.

В окне "Сетевые операции" показаны приложения и службы, запускавшиеся с момента запуска клиента.

Как настроить параметры для отдельного приложения

  1. На боковой панели клиента выберите Состояние.

  2. В разделе "Защита от угроз из сети" выберите Параметры > Показать параметры приложений.

  3. В окне "Показать параметры приложений" выберите необходимое приложение и нажмите кнопку Настроить.

  4. В поле Надежные IP-адреса приложения окна "Настроить параметры приложения" введите IP-адрес или диапазон IP-адресов.

  5. В поле "Порты удаленного сервера" или "Группа локальных портов" выберите порт TCP или UDP.

  6. Для того чтобы указать направление трафика, выберите один из следующих параметров или оба параметра:

    • Для того чтобы разрешить исходящий трафик, выберите Разрешить исходящие соединения.

    • Для того чтобы разрешить входящий трафик, выберите Разрешить входящие соединения.

  7. Для применения правила во время работы экранной заставки выберите Разрешить во время работы экранной заставки.

  8. Для настройки расписания в период, когда действуют (или не действуют) ограничения, нажмите Включить расписание.

  9. Выберите один из следующих вариантов:

    • Для того чтобы указать время, в течение которого действуют ограничения, выберите В течение следующего периода.

    • Для того чтобы указать время, в течение которого не действуют ограничения, выберите За исключением следующего периода.

  10. Настройте расписание.

  11. Нажмите кнопку ОК.

  12. В окне Показать параметры приложений щелкните правой кнопкой мыши на приложении и выберите Разрешить или Заблокировать.

  13. Нажмите кнопку ОК.

Как остановить приложение или службу

  1. На боковой панели клиента выберите Состояние.

  2. В разделе "Защита от угроз из сети" выберите Параметры > Показать сетевые операции.

  3. В поле "Запущенные приложения" щелкните правой кнопкой на приложении и выберите Прервать.

  4. Для подтверждения нажмите Да, затем кнопку Закрыть.
