Параметры уведомлений о сканировании

Для сканирований автоматической защиты или выборочных сканирований пользователь может настроить уведомление и параметры исправления.

Администратор может заблокировать некоторые из этих параметров для клиентов, управляемых из центрального расположения.

Табл.: Параметры уведомления и исправления

Параметр

Описание

Параметры обнаружения

Предусмотрены следующие параметры:

  • Показывать сообщение при обнаружении угрозы

    Указывает, следует ли показывать уведомление на зараженном компьютере в случае, когда во время автоматической защиты или выборочного сканирования была обнаружена угроза безопасности.

    Пользователь может изменить тип информации, показываемой в уведомлении. Можно использовать стандартный текст либо удалить его и ввести более подходящий. Меню, появляющееся при щелчке правой кнопкой мыши в текстовом поле, позволяет вставлять поля переменных в текст сообщения. Такие переменные будут автоматически заменяться на подходящий текст при выводе уведомления.

    В Табл.:  Переменные сообщения представлены переменные сообщений по умолчанию и переменные, которые можно добавить.

  • Показывать окно результатов автоматической защиты

    Указывает, следует ли показывать результаты на зараженных компьютерах. Этот параметр применяется только для автоматической защиты файловой системы.

Параметры исправления

Клиенту может потребоваться прервать процесс или остановить службу для исправления угрозы.

Предусмотрены следующие параметры:

  • Автоматически прерывать процессы

    Если включен этот параметр, клиент прерывает процессы автоматически. Если параметр выключен, Symantec Endpoint Protection спрашивает пользователя, прежде чем предпринять действие над процессом.

  • Автоматически останавливать службы

    Если включен этот параметр, клиент останавливает службы автоматически. Если параметр выключен, клиент спрашивает пользователя, прежде чем предпринять действие над службой.

Примечание:

Пользователи всегда уведомляются о необходимости перезапуска. Это позволяет сохранить данные и закрыть приложения либо отложить перезагрузку.
