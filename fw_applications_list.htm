Обзор параметров приложения

Список приложений содержит приложения и службы, обращавшиеся к сетевому соединению. Приложение добавляется в список, если клиент предлагает пользователю разрешить или блокировать его.

Также можно добавить приложение, если щелкнуть правой кнопкой мыши в окне Сетевые операции и выбрать пункт Разрешить или Блокировать.

См. Сетевые операции.

Для приложения можно настроить ограничения, например указав IP-адреса и порты приложения. Для отмены установленных для приложения ограничений удалите приложение из списка. Когда приложение пытается снова подключиться к сети, появляется сообщение, предлагающее разрешить или блокировать приложение.

Примечание:

Если параметры, настроенные для приложения в этом списке, противоречат правилу брандмауэра, то более приоритетным считается правило брандмауэра.

Параметры приложения

Показывает имя файла, номер версии, действие и путь к приложению или службе.

Клиент можно настроить для выполнения одного из следующих действий с приложением:

  • Разрешить

    Предоставляет приложению полный доступ к сети.

  • Блокировать

    Запрещает приложению доступ к компьютеру или сети.

Настроить

Позволяет настроить дополнительные параметры для выбранного приложения.
