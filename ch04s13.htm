Сведения об исключении объектов из сканирований

Исключениями являются угрозы безопасности, файлы, расширения файлов, процессы, которые следует исключить из сканирования. Если вы сканировали компьютер и знаете, что определенные файлы не представляют угрозу, их можно исключить. Иногда исключения позволяют сократить время сканирования и увеличить производительность системы. Обычно никакие исключения создавать не требуется.

На централизованно управляемых клиентах администратор может создавать исключения из сканирований пользователей. Если настроенное пользователем исключение конфликтует с исключением, заданным администратором, то будет применяться исключение, заданное администратором.

Исключения из списка угроз безопасности

Из угроз безопасности можно исключить:

  • Известные угрозы безопасности

  • Файлы

  • Папки

  • Расширения

В настройках клиента администратор может запретить исключение любого из указанных объектов из сканирования.

Исключения из превентивного сканирования TruScan

Из Превентивного сканирования TruScan могут быть исключены процессы. Для процесса, обнаруженного Превентивным сканированием TruScan, можно указать другое действие. Можно указать принудительное обнаружение процесса.

В настройках клиента администратор может запретить исключение процесса из сканирования.

См. Исключение процесса из превентивного сканирования TruScan.

Исключения из защиты от изменений

Файлы можно исключить из защиты от изменений.

Защита от изменений блокирует попытки любых процессов, кроме Symantec, повлиять на процессы Symantec.
