Планирование запуска сканирования по запросу или при включении компьютера

Плановые сканирования можно дополнить автоматическими сканированиями при включении системы или входе в систему. Часто сканирования при запуске ограничиваются наиболее важными папками, такими как папка Windows и папки с шаблонами Microsoft Word и Microsoft Excel.

В клиенте предусмотрено сканирование при запуске компьютера, которое называется "Автоматически созданное активное сканирование". Это сканирование проверяет наиболее часто подвергающиеся заражению области компьютера при каждом входе пользователя в систему. Это сканирование можно изменить так же, как и любое другое сканирование по запросу. Однако в его параметрах нельзя выключить сканирование загруженных в память файлов и возможных мест заражения.

Если вы часто сканируете один и тот же набор файлов и папок, рекомендуется создать сканирование по запросу, в котором будут перечислены сканируемые элементы. Этот тип осмотра позволяет быстро проверить заданный набор файлов и папок на наличие вирусов и других угроз. Сканирование по запросу следует запускать вручную.

Если существует несколько сканирований при запуске, они будут выполняться последовательно в порядке создания. В настройках клиента администратор может запретить сканирование при запуске.

См. Немедленное сканирование компьютера.

Для просмотра подробных сведений о параметрах в каждом окне нажмите кнопку Справка.

Как запланировать запуск сканирования по запросу или при включении компьютера

  1. На боковой панели клиента выберите Просканировать на наличие угроз.

  2. Выберите Создать сканирование.

  3. Выполните шаги с 3 по 7 для создания запланированного сканирования.

    См. Создание расписания пользовательского сканирования.

  4. В окне Создать сканирование - Когда выполнять сканирование выполните одно из следующих действий:

    • Выберите При запуске.

    • Выберите По запросу.

  5. Нажмите кнопку Далее.

  6. В окне Создать новое сканирование - Имя сканирования введите имя и описание сканирования.

    Например, сканирование можно назвать так: MyScan1

  7. Нажмите кнопку Готово.
