Устранение неполадок: Управление

На этой панели можно просмотреть информацию о клиенте и его защите. Также можно управлять файлом связи, файлом политики и файлом устранения неполадок. Администратор может попросить пользователя изложить информацию на этой панели.

Примечание:

Используйте ее только по указанию администратора или специалистов службы поддержки Symantec.

Табл.: Сведения об управлении

Параметр

Описание

Общие сведения

Отображает следующие сведения о клиенте:

  • IP-адрес или имя DNS сервера. Если сервер подключен, то отображается адрес. Если клиент является управляемым, но в данный момент не подключен к серверу, то в поле будет отображаться состояние Отключен. Если клиент управляется автономно, то для него будет отображаться состояние Автономное управление.

    См. О клиентах, управляемых централизованно, и клиентах под автономным управлением.

  • Группа, в которую входит клиент и из которой он получает защиту.

  • Серийный номер политики безопасности.

  • Получает ли клиент описания вирусов и другие обновления защиты с сервера.

Параметры связи

Позволяет импортировать файл связи таким образом, чтобы преобразовать автономно управляемый клиент в клиент с централизованным управлением.

Профайл политики

Позволяет обновлять, импортировать или экспортировать файл политики на клиенте с автономным управлением. Файл политики содержит параметры защиты компьютера пользователя.

Данные для устранения неполадок

Позволяет сохранить информацию о клиенте и компьютере в текстовом файле. В файле содержится общая информация о политиках, данные о номерах версий файлов описания и другая информация. Этот файл можно отправить по электронной почте администратору службы поддержки Symantec для помощи в решении проблем с клиентом или компьютером.
