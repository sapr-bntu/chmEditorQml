Управление карантином

Файлы помещаются в карантин одним из следующих способов:

  • Клиент перемещает в карантин зараженные объекты, обнаруженные автоматической защитой файловой системы или процедурой сканирования.

  • Файл может быть вручную помещен в карантин пользователем.

По умолчанию функция автоматической защиты и другие виды сканирований при обнаружении зараженного файла пытаются удалить из него вирус. Если исправить файл не удается, он помещается в карантин. При обнаружении угроз безопасности зараженные файлы по умолчанию помещаются в карантин, после чего устраняются побочные эффекты угрозы.

Как вручную поместить файл в карантин

  1. На боковой панели клиента выберите Просмотреть карантин.

  2. Нажмите кнопку Добавить.

  3. Выберите файл, который необходимо поместить в карантин, и нажмите кнопку Добавить.
