Дополнительные параметры почты Интернета

Пользователь может задать дополнительные параметры сканирования почты Интернета.

Табл.: Дополнительные параметры почты Интернета

Параметр

Описание

Параметры сжатых файлов

Позволяет сканировать содержимое сжатых файлов, вложенных в сообщения почты в сети Интернет. Кроме того, можно задать уровень вложенности сканирования сжатых файлов.

Параметры соединения

Включает следующие параметры:

  • Сервер входящей почты (POP3): Порт, на который поступают входящие сообщения.

  • Сервер исходящей почты (SMTP): Порт, на который поступают исходящие сообщения.

  • Разрешить зашифрованные соединения POP3: Позволяет почтовому клиенту получать сообщения по зашифрованному соединению.

  • Разрешить зашифрованные соединения SMTP: Позволяет почтовому клиенту отправлять сообщения по зашифрованному соединению.

    Примечание:

    Клиент не может сканировать сообщения, принимаемые или отправляемые по зашифрованным соединениям.

Эвристика для почтовых червей

Включает следующие параметры:

  • Эвристика для отправляемых сообщений: Этот параметр позволяет указать, следует ли искать в отправляемых сообщениях признаки подозрительного поведения.

  • Первое действие: Выберите первое действие, которое необходимо предпринять при обнаружении подозрительного поведения. Возможные варианты: поместить угрозу в карантин, удалить угрозу, занести сведения в журнал и не выполнять никаких действий.

  • Если первое действие не выполнено: Выберите действие, которое необходимо выполнить над угрозой, если не удалось выполнить первое настроенное действие. Возможные варианты: удалить угрозу или занести сведения в журнал и не выполнять никаких действий.

    Примечание:

    Если в качестве первого действия выбрано Оставить как есть (занести в журнал), то второе действие указать нельзя.

Уведомления о ходе выполнения

Включает следующие параметры:

  • Показывать индикатор состояния при отправке письма: Отображает всплывающее окно состояния выполнения при сканировании сообщений.

  • Показывать значок в области уведомлений: Отображает значок в области уведомлений Windows при сканировании сообщений.
