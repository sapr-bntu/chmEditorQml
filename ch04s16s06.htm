Автоматическое удаление файлов из карантина

Программа может автоматически удалять объекты из карантина через заданный интервал времени. Кроме того, клиент может удалять объекты в случае, если размер папки, в которой они хранятся, достигнет определенного размера. Это позволяет предотвратить накопление файлов, не удаленных вовремя пользователем.

См. Удаление файлов из карантина.

Как настроить автоматическое удаление файлов

  1. На боковой панели клиента выберите Просмотреть карантин.

  2. Выберите Параметры очистки.

  3. В окне "Параметры очистки" выберите одну из следующих вкладок:

    • Объекты, помещенные в карантин

    • Копии зараженных файлов

    • Исправленные файлы

  4. Включите или выключите переключатель Время хранения превышает, чтобы разрешить или запретить клиентам удалять файлы по истечении указанного времени.

  5. Если выбран параметр Время хранения превышает, введите продолжительность времени или выберите его из списка.

  6. Выберите единицы изменения в выпадающем списке. Значение по умолчанию составляет 30 дней.

  7. Если выбран параметр Общий размер папки превышает, укажите максимальный допустимый размер папки (в мегабайтах). Значение по умолчанию составляет 50 мегабайт.

    Если выбраны оба переключателя, то сначала будут удаляться файлы, которые хранятся дольше указанного времени. Если после этого размер папки по-прежнему будет превышать указанное ограничение, то клиент будет по очереди удалять самые старые файлы. Старые файлы будут удаляться до тех пор, пока размер папки не уменьшится до допустимого.

  8. Повторите шаги 4 - 7 на других вкладках.

  9. Нажмите кнопку OK.
