Добавить или изменить правило брандмауэра: Приложения

Можно указать приложение, для которого действует правило брандмауэра. Например, если на компьютере требуется разрешить просмотр веб-страниц с помощью Internet Explorer, то добавьте в список приложение iexplore.exe. Добавьте все приложения, которые используют сетевое соединение и часто применяются.

Табл.: Параметры приложения

Параметр

Описание

Показать только выбранные приложения

Показывает только приложения, отмеченные в столбце Имя файла.

Эти приложения активируют правило.

Приложения

Имя файла, версия и полное имя исполняемого файла приложения или службы, которые нужно разрешить или блокировать.

В поле Имя файла предусмотрен переключатель. Если включить этот переключатель, то клиент будет разрешать или блокировать приложение. Правило брандмауэра игнорирует приложения, для которых не включен переключатель.
