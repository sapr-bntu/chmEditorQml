Изменения порядка обработки правил брандмауэра

Брандмауэр обрабатывает правила в порядке их следования в списке, сверху вниз. Порядок обработки правил брандмауэра можно изменить, изменив расположение этих правил в списке. Переупорядочение правил влияет на порядок их обработки только в выбранном расположении.

Примечание:

Защита будет надежнее, если в начало списка поместить наиболее точные, а в конец - наиболее общие правила.

См. Добавление правила брандмауэра.

Для изменения порядка обработки правил брандмауэра выполните следующие действия

  1. На боковой панели клиента выберите Состояние.

  2. В разделе Защита от угроз из сети выберите Параметры > Настроить правила брандмауэра.

  3. В окне Настроить правила брандмауэра выберите правило, которое необходимо переместить.

  4. Выполните одно из следующих действий:

    • Для того чтобы выбранное правило обрабатывалось раньше, чем правило, расположенное выше него, щелкните на стрелке вверх.

    • Для того чтобы выбранное правило обрабатывалось позже, чем правило, расположенное под ним, щелкните на стрелке вниз.

  5. После перемещения всех необходимых правил нажмите кнопку OK.
