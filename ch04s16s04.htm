Удаление копий зараженных файлов

Прежде чем пытаться удалить или исправить зараженные файлы, клиент по умолчанию создает их резервную копию. После успешного удаления вируса пользователь должен вручную удалить файл из карантина, поскольку копия в карантине все еще заражена. При необходимости задайте период времени, через который файлы будут удаляться автоматически.

См. Автоматическое удаление файлов из карантина.

Как вручную удалить копии зараженных файлов

  1. На боковой панели клиента выберите Просмотреть карантин.

  2. Выберите один или несколько файлов.

  3. Нажмите кнопку Удалить.
