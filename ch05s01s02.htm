Исключения из превентивного поиска угроз TruScan

Если администратор не заблокировал параметры глобальных исключений, то можно создать исключения для превентивного поиска угроз.

Некоторые глобальные исключения из превентивного поиска угроз могут быть созданы администратором. Эти исключения нельзя изменять.
