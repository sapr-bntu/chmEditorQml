Параметры очистки: Копии зараженных файлов

Это окно позволяет запланировать автоматическое удаление копий зараженных файлов клиентом Symantec Endpoint Protection.

Табл.: Параметры удаления копий зараженных файлов

Параметр

Описание

Время хранения превышает

Отметьте этот переключатель, если копии зараженных файлов должны удаляться через заданный промежуток времени. Затем можно указать количество дней, месяцев и лет, по прошествии которых клиент должен удалить файл.

Общий размер папки превышает

Удаление копий зараженных файлов с помощью этого переключателя позволяет ограничить объем каталога, в котором хранятся файлы. При превышении заданного объема клиент удаляет самые старые файлы в каталоге, пока объем каталога не уменьшится до допустимого.
