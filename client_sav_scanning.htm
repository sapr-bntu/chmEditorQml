Тип сканирования или результаты сканирования Symantec AntiVirus

Если клиентом не обнаружены вирусы в проверяемых файлах, то индикатор выполнения укажет, что сканирование завершено.

Если клиент обнаружил вирусы во время сканирования, то в списке результатов будут показаны сведения о заражении.

Примечание:

Операционные системы на некоторых языках могут неправильно интерпретировать некоторые символы в названиях вирусов. Если символы не удается интерпретировать, они отображаются в уведомлениях как знаки вопроса. Например, в именах некоторых вирусов, выраженных в юникоде, есть двухбайтовые символы. Если клиент работает на компьютере с английской операционной системой, такие символы будут показаны в виде знаков вопроса.

Угроза

Название обнаруженной угрозы.

Щелкнув на имени угрозы, можно просмотреть дополнительные сведения о ней на веб-сайте Symantec Security Response.

Исключить

Для некоторых угроз в этом столбце показан переключатель. Включите его, если клиент должен игнорировать эту угрозу при всех последующих сканированиях.

Примечание:

Это следует делать только в тех случаях, когда вы точно знаете, что объект не является вирусом или угрозой безопасности и не представляет угрозы. Например, это может быть программа показа рекламы, которая нужна для работы другого приложения.

Действие

Действие, которое клиент предпринял относительно угрозы, если оно есть.

Тип угрозы

Категория обнаруженной угрозы.

Исходный путь

Путь к каталогу, в котором клиент обнаружил угрозу.

Компьютер

Имя компьютера, на котором клиент обнаружил угрозу.

Пользователь

Имя пользователя, который работал в системе, когда клиент обнаружил угрозу.

Состояние

Состояние обнаруженного файла.

Текущий путь

Путь к каталогу, в котором находится зараженный файл, если он остался на компьютере.

Основное действие

Первое действие, настроенное для данной угрозы.

Резервное действие

Второе действие, настроенное для данной угрозы.

Источник

Тип сканирования, во время которого была обнаружена угроза.

Описание действия

Объяснение действия, которое клиент предпринял относительно обнаруженной угрозы, если оно есть.

Дата и время

Показывает дату и время, когда клиент обнаружил угрозу.

При обнаружении вируса клиент пытается предпринять указанное пользователем действие. Если указанное действие выполнить не удалось, клиент выполняет резервное действие, выбранное для данного типа вируса и сканирования.

Щелкнув правой кнопкой на названии обнаруженной угрозы, можно открыть список действий, доступных после сканирования. Не все действия доступны для каждой угрозы.

Табл.: Действия

Параметр

Описание

Исключить или Удалить из списка исключений

Определяет, что вирус или угроза будут игнорироваться при последующих сканированиях. Удаление из списка исключений означает, что файл будет просканирован клиентом.

Исправить

Удаляет вирус из зараженного файла.

Если клиент успешно удалит вирус из зараженного файла, то другие действия предпринимать не потребуется. Вирус будет удален с компьютера, и его распространение на другие элементы системы станет невозможным.

Когда клиент очищает файл, он удаляет вирус из зараженного файла, загрузочного сектора или таблицы разделов. Удаленный вирус теряет способность к распространению.

В некоторых случаях исправленный файл может оказаться непригодным к использованию. Это происходит из-за вредоносных действий вируса, а не в результате его удаления из файла.

Некоторые зараженные файлы исправить нельзя.

Удалить окончательно

Удаляет зараженный файл с жесткого диска компьютера.

Это действие следует выбирать только в том случае, если файл можно восстановить из незараженной резервной копии. Удаленный файл нельзя восстановить из Корзины.

Примечание:

Это действие следует с осторожностью применять при выборе действий для угроз безопасности, поскольку удаление файла может привести к потере работоспособности приложения.

Отменить действие

Отменяет действие, которое клиент выполнил по отношению к обнаруженной угрозе.

Отменить действие можно не всегда. Например, клиент не может восстановить удаленный файл.

Поместить в карантин

Выполняет одно из следующих действий:

  • В случае обнаружения вируса клиент перемещает зараженный файл в Карантин. Вирусы из помещенных в карантин зараженных файлов теряют способность к распространению.

  • В случае обнаружения угрозы безопасности перемещает зараженные файлы в карантин и пытается устранить побочные эффекты.

    В карантине хранятся сведения о всех выполненных действиях. С помощью этих сведений можно восстановить состояние компьютера, в котором он находился до того, как клиент удалил удалила угрозу.
