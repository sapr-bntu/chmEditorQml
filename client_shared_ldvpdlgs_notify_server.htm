Отправить сообщение получателю: Почтовый сервер

Для того чтобы уведомить отправителя зараженного сообщения, необходимо указать адрес почтового сервера, с помощью которого это уведомление будет отправлено.

Табл.: Параметры почтового сервера

Параметр

Описание

Почтовый сервер

Имя почтового сервера

Порт

Порт, через который отправляются сообщения

Имя пользователя

Имя пользователя, необходимое для обращения к серверу и отправки сообщений

Пароль

Пароль для почтового сервера

Обратный путь

Сведения об обратном пути для почтового сервера. Обычный путь указывать не обязательно, однако некоторые почтовые серверы требуют это значение. Можно указать доменное имя компьютера, создающего сообщение.
