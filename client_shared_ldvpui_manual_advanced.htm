Дополнительные параметры сканирования

Эти параметры позволяют настроить ход сканирования.

Табл.: Дополнительные параметры сканирования

Дополнительный параметр

Описание

Параметры сжатых файлов

Сканирует содержимое сжатых файлов. Кроме того, можно задать уровень вложенности сканирования сжатых файлов.

Параметры резервного копирования

Автоматически создает резервную копию файла перед попыткой исправить его.

Параметры окна диалога

Определяет, должно ли отображаться окно с информацией о ходе сканирования. Также можно указать, должно ли это окно автоматически закрываться после завершения сканирования.

Параметры настройки

Этот параметр позволяет сбалансировать производительность сканирования и его влияние на другие приложения.

Параметры миграции между разными типами памяти

Позволяет задать настройки сканирования файлов, которые обслуживаются системой Hierarchical Storage Management (HSM) и автономными системами резервного копирования.

Примечание:

Для выбора правильных значений этих параметров проконсультируйтесь с поставщиком HSM или применяемой системы резервного копирования.
