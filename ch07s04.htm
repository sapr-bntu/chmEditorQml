Исправление компьютера

Когда клиент обнаруживает, что требования политики целостности хоста не соблюдены, он реагирует одним из следующих способов:

  • Клиент автоматически загружает обновление программы.

  • Клиент предлагает пользователю загрузить обновление.

Как исправить компьютер

  • В появившемся окне Symantec Endpoint Protection выполните одно из следующих действий:

    • Для того чтобы узнать, какие требования к безопасности не выполнены, нажмите кнопку Сведения.

    • Для того чтобы немедленно установить программу, нажмите кнопку Восстановить сейчас.

      В некоторых случаях начатую процедуру установки можно отменить.

    • Для того чтобы приостановить установку программы, нажмите Напомнить позже и выберите промежуток времени в выпадающем списке.

      Администратор может указать, сколько раз установка может быть приостановлена.
