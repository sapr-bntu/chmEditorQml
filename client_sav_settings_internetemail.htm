защиты от вирусов и программ-шпионов: Автоматическая защита почты в Интернете

Функция автоматической защиты может сканировать вложения почтовых сообщений Интернета.

Администратор может заблокировать некоторые из этих параметров для клиентов, управляемых из центрального расположения.

Примечание:

Для исключения отдельных файлов и каталогов из сканирования можно настроить глобальные исключения.

Табл.: Параметры автоматической защиты почты Интернета

Параметр

Описание

Включить автоматическую защиту почты Интернета

Включает или выключает автоматическую защиту электронной почты в Интернете.

Типы файлов

Указывает, следует ли автоматически сканировать все типы файлов или только файлы с указанными расширениями.

Предусмотрены следующие варианты:

  • Все типы: Сканировать все файлы на компьютере, независимо от их типа.

  • Выбранные: Выберите этот вариант для сканирования файлов с указанными разрешениями.

  • Расширения: Указывает, что следует сканировать только файлы с указанными расширениями. Можно добавить записки для программ и документов, расширения которых отсутствуют в списке. Будут сканироваться только файлы с расширениями из этого списка. Файлы с другими расширениями сканироваться не будут.

Действия

Позволяет настроить действия, которые должны выполняться функцией автоматической защиты при обнаружении макровирусов, иных вирусов и угроз безопасности.

Уведомления

Указывает, должна ли функция автоматической защиты показывать сообщение при обнаружении вируса или угрозы безопасности. Также позволяет задать содержимое сообщения.

Дополнительно

Позволяет настроить дополнительные параметры сканирования, такие как параметры сжатия и соединения.

Почтовые сообщения

Можно настроить следующие типы почтовых сообщений:

  • Добавлять предупреждение в электронное письмо: Добавляет предупреждение в зараженные электронные сообщения. Стандартный текст можно изменить.

  • Послать письмо отправителю: Направляет уведомления отправителям зараженных сообщений в почтовых программах Интернета. Стандартный текст письма можно изменить.

  • Отправлять сообщение другим пользователям: Информирует пользователей по указанным адресам о заражении почты Интернета. Стандартный текст письма можно изменить.
