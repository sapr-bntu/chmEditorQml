Отправить сообщение другим пользователям: Другие

Можно автоматически уведомлять других пользователей об обнаружении вирусов и угроз безопасности.

Укажите адреса электронной почты пользователей, которых необходимо уведомлять о вирусе или угрозе безопасности. Клиент отправит сообщение с информацией о заражении вирусом по всем адресам в списке. Нажмите Добавить, чтобы добавить адреса электронной. почты, либо выберите адреса, которые нужно удалить, и нажмите Удалить.
