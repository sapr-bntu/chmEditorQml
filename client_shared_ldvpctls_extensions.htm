Выбранные расширения

Если во время сканирования должны проверяться только файлы с заданными расширениями, пользователь может добавить такие расширения в список или исключить их.

Табл.: Параметры расширений

Параметр

Описание

По умолчанию

Восстанавливает исходное состояние списка расширений.

Все добавленные пользователем расширения удаляются, а все стандартные расширения добавляются.

Добавить общие типы расширения для

Предусмотрены следующие параметры:

  • Программы: Выбирает все расширения для распространенных программ.

  • Документы: Выбирает все расширения для распространенных документов.
