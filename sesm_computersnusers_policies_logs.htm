Параметры журнала клиента для имя группы

В этом окне можно выполнять следующие задачи:

  • Разрешить передачу содержимого журнала от клиентов диспетчеру

  • Задать размер, срок хранения и параметры амортизатора для кэша передачи на клиентах

Если некоторые сетевые компьютеры не подключаются в течение длительного времени, рекомендуется ограничить размер кэша передачи. Эти параметры применяются ко всем клиентам из группы.

Примечание:

Эти параметры применяются только к тем записям журнала, которые хранятся в кэше передачи. Они не влияют на количество записей или размер журналов на клиентах.

Табл.: Параметры группы для журналов клиентов

Параметр

Описание

Максимальный размер

Задает максимальный размер данных, передаваемых в этом журнале.

Для системного журнала, журнала безопасности, журнала угроз и журнала трафика значение по умолчанию составляет 512 КБ. Для журнала пакетов и журнала управления значение по умолчанию составляет 1024 КБ.

Хранить в течение

Задает количество дней для хранения данных.

Значение по умолчанию составляет 14 дней.

Загрузить на сервер управления

Указывает, что клиенты данной группы должны загружать этот журнал на сервер управления.

Значение по умолчанию - загружать журналы.

Период амортизатора

Задает период времени для сжатия похожих записей журнала в одну.

Значение по умолчанию составляет 7200 секунд (1 час).

Простой амортизатора

Задает максимальный период ожидания клиента между операциями сжатия похожих записей журнала в одну запись.

Значение по умолчанию составляет 10 секунд.

Максимальный размер загружаемых данных

Задает максимальное количество записей, одновременно передаваемых клиентом диспетчеру.

Значение по умолчанию составляет 100 записей.

Просмотреть содержимое журналов клиентов можно с помощью консоли или веб-браузера.

Табл.: Просмотр содержимого журналов клиентов с помощью консоли

Журнал клиента

Источники информации в консоли

Системный журнал

Эту информацию можно просмотреть в Системном журнале операций клиента.

Журналы безопасности и рисков

Эту информацию можно просмотреть в журналах Угроз.

Журналы безопасности

Эту информацию можно просмотреть в журнале Управления приложениями, журнале Проверки хоста на соответствие требованиям и Журнале атак функции защиты от угроз из сети.

Журнал трафика

Эту информацию можно просмотреть в Журнале трафика функции защиты от угроз из сети.

Журнал пакетов

Эту информацию можно просмотреть в Журнале пакетов функции защиты от угроз из сети.

Журнал управления

Эту информацию можно просмотреть в журнале Управления приложениями и в журнале Управления устройствами.
