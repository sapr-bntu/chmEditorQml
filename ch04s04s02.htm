Сведения о сканировании файлов с определенными расширениями

Клиент может сканировать на компьютере только файлы с определенными расширениями.

Можно выбрать один из следующих типов расширений файлов:

Документы

Документы Microsoft Word и Excel, а также файлы шаблонов, связанные с этими документами. Клиент проверяет документы на наличие макровирусов.

Программы

Включают динамически компонуемые библиотеки (.dll), командные файлы (.com), исполняемые файлы (.exe) и другие программные файлы. Клиент проверяет программные файлы на наличие программных вирусов.

Как добавить исключения файлов в список сканируемых файлов функции автоматической защиты

  1. На боковой панели клиента выберите Изменить параметры.

  2. В разделе "Защита от вирусов и программ-шпионов" нажмите кнопку Изменить параметры.

  3. В окне Параметры защиты от вирусов и программ-шпионов откройте вкладку Автоматическая защита файловой системы, раздел Типы файлов и нажмите кнопку Выбранные.

  4. Нажмите кнопку Расширения.

  5. Введите расширение в текстовом поле и нажмите кнопку Добавить.

  6. Повторите шаг 5 необходимое количество раз и нажмите OK.

  7. Нажмите кнопку ОК.

Как добавить расширения файлов в список файлов, проверяемых сканированием по запросу, плановым сканированием или сканированием при запуске

  1. На боковой панели клиента выберите Просканировать на наличие угроз.

  2. Щелкните правой кнопкой мыши на типе сканирования, для которого нужно добавить расширения, и выберите пункт Изменить.

    Изменения будут применены только для выбранного сканирования.

  3. На вкладке Параметры сканирования отметьте пункт Выбранные расширения в разделе Типы файлов и нажмите кнопку Расширения.

  4. Введите расширение и нажмите кнопку Добавить.

  5. Повторите шаг 4 необходимое количество раз и нажмите OK.

  6. Нажмите кнопку ОК.
