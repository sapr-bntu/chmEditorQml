Создать сканирование

Пользователь может настроить выборочное сканирование компьютера, которое будет выполняться по запросу, в указанное время или при запуске.

Табл.: Параметры запуска сканирования

Параметры запуска

Описание

По запросу

Выполнять выбранное сканирование только по запросу.

В указанное время

Выполнять сканирование только в указанное время в указанные дни.

При запуске

Выполнять сканирование при загрузке системы или входе в систему нового пользователя.

Табл.: Параметры объектов для сканирования

Параметр сканирования

Описание

Активный

На наличие вирусов и угроз безопасности сканируется память и другие частые точки заражения компьютера.

Полный

Проверяет весь компьютер на наличие вирусов и угроз безопасности. При таком сканировании проверяется загрузочный сектор, программы в памяти, а также все файлы и папки. Для сканирования сетевых дисков может потребоваться ввести пароль.

Выборочный

Сканирует только выбранные файлы и каталоги.
