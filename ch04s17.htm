Отправка информации об обнаружении угроз в Symantec Security Response

Информация об автоматических сканированиях или частоте обнаружения угроз может автоматически отправляться в Symantec Security Response. Сведения о частоте обнаружения угроз потенциально помогают компании Symantec уточнить описания вирусов. В состав этих сведений включается информация о вирусах и угрозах безопасности, которые выявлялись клиентами чаще всего. Специалисты Symantec Security Response могут удалить сигнатуры, которые не выявляются, и предоставить сегментированный список сигнатур тем клиентам, которым он требуется. Сегментированные списки повышают производительность сканирования.

По умолчанию отправка сведений о частоте обнаружения включена.

Примечание:

Учтите, что параметры отправки могут быть заблокированы администратором.

В компанию Symantec можно также отправить объекты из карантина.

См. Отправка потенциально зараженного файла в группу Symantec Security Response для анализа.

Для отправки информации об обнаружении угроз в Symantec Security Response выполните следующие действия:

  1. На боковой панели клиента выберите Изменить параметры.

  2. В разделе "Защита от вирусов и программ-шпионов" нажмите кнопку Изменить параметры.

  3. На вкладке "Отправка" отметьте переключатель Автоматически отправлять найденные вирусы и угрозы безопасности.

  4. Нажмите кнопку OK.
