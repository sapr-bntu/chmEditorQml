Параметры защиты от вирусов и программ-шпионов: Общие

На клиенте можно настроить общие параметры.

Табл.: Общие параметры

Параметр

Описание

Хранение журнала

Задает период времени, по истечении которого из журналов клиента должны удаляться записи

Этот параметр не влияет на те события, которые сохраняются в журнале с помощью консоли управления. С помощью этого параметра можно сократить размер журналов на компьютере.

Защита веб-браузера

Задает URL домашней страницы, который применяется, если адрес домашней страницы был подменен угрозой безопасности.

Клиент применяет этот URL в процессе устранения угрозы.
