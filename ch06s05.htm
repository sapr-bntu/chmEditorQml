Добавление правила брандмауэра

При добавлении правил важно понимать, какого эффекта необходимо добиться. Например, можно разрешить весь трафик от конкретного источника или заблокировать пакеты UDP с веб-сайта.

См. Сведения о правилах брандмауэра.

См. Сведения об элементах правила брандмауэра.

Для добавления правила брандмауэра

  1. На боковой панели клиента выберите Состояние.

  2. В разделе Защита от угроз из сети выберите Параметры > Настроить правила брандмауэра.

  3. В окне Настроить правила брандмауэра нажмите кнопку Добавить.

  4. На вкладке Общие введите имя правила, затем выберите Блокировать этот трафик или Разрешить этот трафик.

  5. Для того чтобы указать триггеры для правила, выберите одну из следующих вкладок:

    • Хосты

    • Порты и протоколы

    • Приложения

    Для просмотра подробных сведений параметрах на каждой вкладке нажмите кнопку Справка.

  6. Для того чтобы указать период времени, в течение которого правило должно быть активно или неактивно, выберите Включить расписание на вкладке Расписание, затем настройте расписание.

  7. После внесения всех необходимых изменений нажмите кнопку OK.

  8. В окне Настроить правила брандмауэра убедитесь, что напротив правила в списке Имя правила стоит отметка.

    Можно также изменить порядок обработки правил брандмауэром.

    См. Изменения порядка обработки правил брандмауэра.

  9. Нажмите кнопку OK.
