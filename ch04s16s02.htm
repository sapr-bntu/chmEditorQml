Повторное сканирование файлов в карантине на наличие вирусов

После помещения файла в карантин обновите описания. После обновления описаний файлы, находящиеся в карантине, могут быть просканированы, исправлены и восстановлены автоматически. Если появилось окно мастера исправления, можно повторно просканировать файлы, находящиеся в карантине.

См. Повторное сканирование файлов вручную.

Если после повторного сканирования клиенту не удалось удалить вирус, зараженный файл можно отправить в Symantec Security Response для анализа.

См. Отправка потенциально зараженного файла в группу Symantec Security Response для анализа.

Как повторно просканировать файлы в карантине с помощью Мастера исправления

  1. Когда появится окно мастера исправления, нажмите кнопку Да.

  2. Нажмите Далее.

  3. Выполняйте появляющиеся на экране указания по повторному сканированию содержимого карантина.
