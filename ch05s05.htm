Отправка информации о превентивном сканировании TruScan в Symantec Security Response

По умолчанию информация о процессах, найденных во время превентивного сканирования, передается в Symantec Security Response. Symantec анализирует полученную информацию и определяет, идет ли речь о реальной угрозе. Если да, то Symantec создает сигнатуру для обнаружения этой угрозы. Эта сигнатура добавляется в новую версию описаний.

Отправляемая информация о процессе включает в себя следующие сведения:

  • Путь к исполняемому файлу

  • Исполняемый файл

  • Информацию о файле и записи загрузки реестра, ссылающиеся на угрозу

  • Информацию о внутреннем состоянии

  • Текущую версию данных, применяемых превентивным поиском угроз

Никакая конфиденциальная информация, позволяющая идентифицировать компьютер, не отправляется.

Отправка результатов превентивного поиска угроз в Symantec Security Response включена по умолчанию.

Примечание:

Параметры отправки могут быть заблокированы администратором.

Для получения дополнительных сведений о параметрах, участвующих в этой процедуре, нажмите кнопку "Справка".

Отправка информации о превентивном сканировании TruScan в Symantec Security Response

  1. На боковой панели клиента выберите Изменить параметры.

  2. В разделе "Защита от вирусов и программ-шпионов" нажмите кнопку Изменить параметры.

  3. В окне "Параметры защиты от вирусов и программ-шпионов" откройте вкладку "Отправка" и включите или отключите переключатель Автоматически отправлять найденное в ходе превентивного сканирования TruScan.

  4. Нажмите кнопку OK.
