Параметры защиты от вирусов и программ-шпионов: Отправка

Пользователь может указать, следует ли отправлять информацию в Symantec Security Response.

Значение этого параметра для управляемых клиентов может быть заблокировано администратором.

Табл.: Параметры отправки

Параметр

Описание

Автоматически отправлять найденные вирусы и угрозы безопасности

Указывает, должны ли компьютеры клиентов отправлять сведения о частоте заражения в Symantec.

С помощью этих сведений специалисты Symantec Security Response могут выяснить, какие описания вирусов более не требуются.

Автоматически отправлять найденное в ходе превентивного сканирования TruScan

Указывает, должен ли компьютер отправлять сведения о процессах, выявленных при превентивном сканировании TruScan.

Такие сведения помогают компании Symantec бороться с новыми угрозами.
