Параметры журнала отладки Управления клиентами

С помощью этого окна можно настроить журнал отладки для сбора информации о следующих операциях клиента:

  • Соединение с сервером

  • События брандмауэра

  • Переключение расположения

  • Соответствие целостности хоста требованиям

  • Задачи EAP 802.1x

Эту информацию следует передать администратору или службе поддержки Symantec. Файл журнала - это текстовый файл с именем debug.log.

Предупреждение!

При включении журнала отладки размер журнала может увеличиваться очень быстро. Следовательно, параметр можно задавать только по указанию администратора или специалистов службы поддержки Symantec.

Табл.: Параметры журнала отладки Управления клиентами

Параметр

Описание

Включить отладку

Включает журнал для регистрации данных.

Уровень отладки

Регистрирует события сканирования. Значение по умолчанию - 0. Обычно это значение рекомендуется использовать в целях устранения неполадок.

Уровень ведения журнала

Регистрирует события брандмауэра. В журнале регистрируется информация о заблокированных пакетах, включая применяемое правило и приложение, отправившее пакет. Значение по умолчанию - 0, оно указывает на регистрацию всех возможных событий.

Размер файла журнала (КБ)

Увеличивает или уменьшает размер файла журнала. Размер журнала по умолчанию - 256 КБ. Возможно, в целях экономии дискового пространства, потребуется задать меньший размер файла.
