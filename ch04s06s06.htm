Настройка распознавания типов файлов в функции автоматической защиты

По умолчанию автоматическая защита сканирует файлы всех типов. Для ускорения сканирования можно проверять только файлы с определенными расширениями.

Например, можно сканировать только файлы со следующими расширениями:

  • .exe

  • .com

  • .dll

  • .doc

  • .xls

Обычно вирусы заражают только определенные типы файлов. Однако выбор только некоторых расширений для сканирования в рамках автоматической защиты ослабляет защиту компьютера. В список расширений по умолчанию включены файлы, вероятность заражения которых вирусами наиболее велика.

Функция автоматической защиты сканирует все исполняемые файлы, а также все файлы .exe и.doc. Она правильно определяет тип файла даже в том случае, если расширение файла было изменено вирусом. Например, она будет сканировать файлы .doc, даже если их расширение было изменено вирусом.

Для обеспечения максимально надежной защиты компьютера от вирусов и угроз безопасности функция автоматической защиты должна сканировать все типы файлов.

Как настроить распознавание типов файлов в функции автоматической защиты

  1. На боковой панели клиента выберите Изменить параметры.

  2. В разделе "Защита от вирусов и программ-шпионов" нажмите кнопку Изменить параметры.

  3. Откройте вкладку "Автоматическая защита" и выполните одно из следующих действий в разделе "Типы файлов":

    • Выберите Все типы для сканирования всех файлов.

    • Выберите пункт Выбранные, чтобы сканировались только файлы с расширениями из заданного списка, а затем нажмите кнопку Расширения для изменения списка расширений сканируемых файлов.

  4. Если был отмечен пункт "Выбранные", то выберите или отмените выбор параметра Определять типы файлов по содержимому.

  5. Нажмите кнопку OK.
