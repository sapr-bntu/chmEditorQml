Параметры защиты от вирусов и программ-шпионов: Автоматическая защита Notes

Функция автоматической защиты может сканировать вложения сообщений Lotus Notes.

Администратор может заблокировать некоторые из этих параметров для управляемых клиентов.

Примечание:

Для исключения отдельных файлов и каталогов из сканирования можно настроить глобальные исключения.

Табл.: Параметры автоматической защиты Lotus Notes

Параметр

Описание

Включить автоматическую защиту Lotus Notes

Включает или выключает автоматическую защиту Lotus Notes

Параметр недоступен, если на клиентском компьютере не установлен Lotus Notes.

Типы файлов

Указывает, следует ли автоматически сканировать все типы файлов или только файлы с указанными расширениями.

Предусмотрены следующие варианты:

  • Все типы: Сканировать все файлы независимо от их типа.

  • Выбранные: Выберите этот вариант для сканирования файлов с указанными разрешениями. Нажмите кнопку "Расширения", чтобы добавить программы и документы, расширения которых отсутствуют в списке. При необходимости можно восстановить список расширений по умолчанию.

Расширения

Указывает, что следует сканировать только файлы с указанными расширениями.

Список расширений можно изменять по своему усмотрению. Будут сканироваться только файлы с расширениями из этого списка. Файлы с другими расширениями сканироваться не будут.

Примечание:

Для исключения файлов или каталогов из сканирования создайте глобальное исключение. Эти исключения действуют во всех операциях сканирования функции защиты от вирусов и программ-шпионов.

Дополнительно

Позволяет настроить дополнительные параметры сканирования, такие как параметры сжатия и соединения.

Действия

Позволяет настроить действия, которые должны выполняться функцией автоматической защиты при обнаружении макровирусов, иных вирусов и угроз безопасности.

Уведомления

Указывает, должна ли функция автоматической защиты показывать сообщение при обнаружении вируса или угрозы безопасности.

Также можно указать содержимое сообщения.

Почтовые сообщения

Можно настроить следующие типы почтовых сообщений:

  • Добавлять предупреждение в электронное письмо: Добавляет предупреждение в зараженные электронные сообщения. Стандартный текст можно изменить.

  • Послать письмо отправителю: Информирует отправителей зараженных сообщений. Стандартный текст письма можно изменить.

  • Отправлять письмо указанным получателям: Информирует о заражении пользователей по указанным адресам. Стандартный текст письма можно изменить.
