Исключение объектов из сканирования

Исключение можно создать на странице Изменить параметры. Исключения можно настраивать и в процессе создания или изменения сканирований по запросу, а также при изменении параметров автоматической защиты.

См. Сведения об исключении объектов из сканирований.

Исключения действуют во всех сканированиях. Даже те исключения, которые были настроены в процессе создания или изменения отдельного сканирования, действуют во всех сканированиях.

Примечание:

В Windows Server 2008 Server Core вид окон может отличаться от приведенных в этой главе.

Для просмотра подробных сведений о параметрах в каждом окне нажмите кнопку Справка.

Как исключить угрозу безопасности из сканирования

  1. На боковой панели клиента выберите Изменить параметры.

  2. В разделе Глобальные исключения нажмите кнопку Изменить параметры.

  3. В окне Глобальные исключения откройте вкладку Исключения пользователя и нажмите Добавить > Исключение из списка угроз безопасности > Известные угрозы.

  4. В окне Добавить исключения из списка известных угроз безопасности выберите угрозы безопасности, которые следует исключить из сканирований.

  5. Если при обнаружении исключенной угрозы безопасности следует заносить запись в журнал, то включите переключатель Добавлять в журнал запись об обнаружении угрозы.

  6. Нажмите кнопку ОК.

  7. В окне Глобальные исключения нажмите кнопку Закрыть.

Как исключить файл или папку из сканирования

  1. На боковой панели клиента выберите Изменить параметры.

  2. В разделе "Глобальные исключения" нажмите кнопку Изменить параметры.

  3. В окне Глобальные исключения откройте вкладку Исключения пользователя и нажмите Добавить > Исключение из списка угроз безопасности.

  4. Выполните одно из следующих действий:

    • Выберите Файл. В списке Добавить исключения угроз безопасности по имени файла выберите файл для исключения и нажмите кнопку Добавить.

    • Выберите Папка. В окне Добавить исключения угроз безопасности по имени папки выберите папку, включите или выключите Включая подпапки и нажмите кнопку Добавить.

  5. В окне Глобальные исключения нажмите кнопку Закрыть.
