Параметры очистки: Исправление элементов

Это окно позволяет запланировать автоматическое удаление исправленных файлов клиентом Symantec Endpoint Protection.

Табл.: Параметры удаления исправленных элементов

Параметр

Описание

Время хранения превышает

Отметьте этот переключатель, если исправленные элементы должны удаляться через заданный промежуток времени. Затем можно указать количество дней, месяцев и лет, по прошествии которых клиент должен удалить файл.

Общий размер папки превышает

Удаление исправленных файлов с помощью этого переключателя позволяет ограничить объем каталога, в котором хранятся исправленные файлы. При превышении заданного объема клиент удаляет самые старые файлы в каталоге, пока объем каталога не уменьшится до допустимого.
